# ✅ ДОБАВЛЕН БЛОК: Отображение выходных дней в ScheduleCard

## 🎯 Что добавлено

Теперь в карточке расписания врача отображается список предстоящих выходных дней с возможностью их удаления.

## 🔧 Новая функциональность

### 1. ✅ Автоматическое отображение выходных дней
- Блок появляется **только если есть выходные дни**
- Если выходных дней нет - блок **невидимый**
- Показывает ближайшие 30 дней

### 2. ✅ Красивое оформление каждого выходного дня
- **Дата** в локальном формате (дд.мм.гггг)
- **Причина** (если указана)
- **Иконка календаря** для визуального оформления
- **Кнопка удаления** для быстрого удаления

### 3. ✅ Интерактивность
- Hover эффекты при наведении
- Кнопка удаления с подтверждением
- Автоматическое обновление списка после изменений

### 4. ✅ Адаптивность
- Максимальная высота с прокруткой (`max-h-48 overflow-y-auto`)
- Красивые отступы и скругления
- Responsive дизайн

## 🎨 Дизайн блока

### Структура каждого элемента:
```jsx
<div className="bg-[#F5F8FF] border border-blue-100 rounded-[16px] p-4">

```

### Цветовая схема:
- **Фон**: `#F5F8FF` (светло-голубой)
- **Граница**: `border-blue-100`
- **Иконка**: голубой круг с календарем
- **Удаление**: красная кнопка с корзиной

## 🔗 Интеграция с API

### Получение данных:
```typescript
const { data: daysOff = [], refetch } = useDaysOff(
    today.toISOString().split('T')[0],
    futureDate.toISOString().split('T')[0]
);
```

### Удаление выходного дня:
```typescript
const handleDeleteDayOff = async (dayOffId: number) => {
    await deleteDayOff.mutateAsync(dayOffId);
    refetch(); // Обновляем список
};
```

## 🎯 Логика отображения

### Условное отображение:
```jsx
{daysOff.length > 0 && (
    <div className="mt-6">
        {/* Блок с выходными днями */}
    </div>
)}
```

### Если выходных дней нет:
- Блок полностью скрыт
- Показывается только кнопка "Добавить"

### Если есть выходные дни:
- Показывается заголовок "Предстоящие выходные дни"
- Список всех выходных дней
- Каждый день можно удалить
