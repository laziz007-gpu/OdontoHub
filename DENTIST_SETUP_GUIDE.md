### Способ 2: Регистрация врачей через приложение

1. Откройте приложение
2. Перейдите на страницу регистрации
3. Выберите роль "Врач"
4. Заполните данные:
   - Полное имя
   - Номер телефона
   - Email (необязательно)
5. Нажмите "Зарегистрироваться"

Врач будет добавлен в базу данных и появится в списке для пациентов.

## Как это работает

### Для пациентов (Local Mode):

1. Пациент регистрируется → устанавливается флаг `is_first_time`
2. Перенаправление на `/doctors` вместо `/home`
3. Показывается приветственный баннер на 5 секунд
4. Отображается список врачей из базы данных
5. Пациент может нажать "Записаться" на любого врача

### Для врачей (API Mode):

Когда backend запущен:
- Frontend делает запрос к `GET /dentists/`
- Backend возвращает всех врачей из таблицы `dentist_profiles`
- Врачи отображаются с их данными:
  - Имя
  - Специализация
  - Клиника
  - Адрес
  - Часы работы
  - Контакты (Telegram, Instagram, WhatsApp)

## Проверка

После добавления врачей:

1. Зарегистрируйтесь как новый пациент
2. Вы должны увидеть:
   - Синий баннер "Добро пожаловать! 👋"
   - Список врачей из базы данных
   - Кнопку "Записаться" на каждой карточке врача
3. Нажмите "Записаться" → запись добавится в localStorage
4. Перейдите в "Приёмы" → увидите созданную запись

## Структура данных врача

```typescript
{
  id: number;
  user_id: number;
  full_name: string;
  specialization: string;  // "Терапевт", "Ортодонт", и т.д.
  clinic: string;          // Название клиники
  address: string;         // Адрес
  work_hours: string;      // "09:00-18:00"
  phone: string;
  telegram: string;
  instagram: string;
  whatsapp: string;
  verification_status: "approved" | "pending" | "rejected";
}
```

## Troubleshooting

### Врачи не отображаются?

1. Проверьте, что backend запущен: `uvicorn app.main:app --reload`
2. Проверьте консоль браузера на ошибки
3. Убедитесь, что в `.env` файле: `VITE_API_URL=http://localhost:8000`
4. Проверьте базу данных: запустите `create_test_dentists.py` еще раз

### Local Mode vs API Mode

- **Local Mode**: Когда token начинается с `local_token_`, данные берутся из localStorage
- **API Mode**: Когда backend запущен, данные берутся из базы данных через API

В Local Mode врачи все равно будут загружаться из API, если backend запущен!
