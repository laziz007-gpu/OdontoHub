## 🎨 Как теперь работает

1. **Пользователь кликает кнопку "Qo'shish"** в разделе выходных дней
2. **Открывается модальное окно** с календарем и полем для причины
3. **Пользователь выбирает дату** (только будущие даты доступны)
4. **Указывает причину** (например: "Поход в горы", "Личные дела")
5. **Нажимает "Qo'shish"** для сохранения
6. **Система отправляет запрос** на бэкенд API
7. **Показывается уведомление** об успехе или ошибке

## 🔗 Интеграция с бэкендом

Фронтенд теперь полностью интегрирован с бэкенд API:
- `POST /api/days-off/` - добавление выходного дня
- `GET /api/days-off/` - получение списка выходных дней
- `DELETE /api/days-off/{id}` - удаление выходного дня
