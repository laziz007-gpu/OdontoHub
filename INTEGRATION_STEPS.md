## 🚀 Интеграция (3 шага)

### Шаг 1: Добавить роут
Откройте `src/Routes/index.tsx` и добавьте:

```tsx
import AdminDentists from '../Pages/AdminDentists';

// В ваших роутах добавьте:
<Route path="/admin/dentists" element={<AdminDentists />} />
```

### Шаг 2: Добавить в меню (опционально)
Добавьте ссылку в ваше админское меню:

```tsx
<Link to="/admin/dentists">Управление врачами</Link>
```

### Шаг 3: Готово!
Откройте `/admin/dentists` в браузере и пользуйтесь!

## 🎯 Что получите

### Функции админки:
- 📋 Просмотр всех врачей
- ✅ Одобрение/отклонение врачей
- 🗑️ Удаление врачей
- 🔄 Обновление списка
- 📱 Адаптивный дизайн

### Backend endpoints (уже работают):
- `GET /dentists/list` - список врачей
- `PUT /dentists/{id}/status` - изменить статус
- `DELETE /dentists/{id}` - удалить врача
