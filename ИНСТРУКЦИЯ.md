# Быстрая Инструкция - Деплой Backend

## Проблема
Вы пытались запустить `deploy_backend.bat` из неправильной папки.

## Решение - 2 простых способа

---

## Способ 1: Автоматический (PowerShell скрипт)

### Запустите из любой папки:

```powershell
powershell -ExecutionPolicy Bypass -File "D:\OdontoHUB\WebApp\OdontoHub-1\create_migration_files.ps1"
```

Скрипт:
- Создаст файл миграции
- Обновит render.yaml
- Покажет следующие шаги

Затем просто:
```powershell
cd "D:\OdontoHUB\WebApp\Backend-new fix\Backend"
git add .
git commit -m "Add dentist profile fields migration"
git push origin master
```

---

## Способ 2: Ручной (Самый простой!)

### Шаг 1: Зайдите в Render Dashboard
1. Откройте: https://dashboard.render.com
2. Найдите вашу базу данных `odontohub-db`
3. Нажмите "Shell"

### Шаг 2: Выполните SQL
Скопируйте и вставьте эти 3 строки:

```sql
ALTER TABLE dentist_profiles ADD COLUMN IF NOT EXISTS age INTEGER;
ALTER TABLE dentist_profiles ADD COLUMN IF NOT EXISTS experience_years INTEGER;
ALTER TABLE dentist_profiles ADD COLUMN IF NOT EXISTS works_photos TEXT;
```

### Шаг 3: Запушьте код
```powershell
cd "D:\OdontoHUB\WebApp\Backend-new fix\Backend"
git add .
git commit -m "Update backend"
git push origin master
```

**Готово!** ✅

---

## Что выбрать?

- **Способ 1** - если хотите автоматическую миграцию при каждом деплое
- **Способ 2** - если хотите быстро исправить и задеплоить (рекомендуется!)
