## 🚀 Как использовать

### Шаг 1: Добавить в роутинг
Добавьте в `src/Routes/index.tsx`:

```tsx
import AdminDentists from '../Pages/AdminDentists';

// В ваших роутах:
<Route path="/admin/dentists" element={<AdminDentists />} />
```

### Шаг 2: Добавить в меню
Добавьте ссылку в админское меню:

```tsx
<Link to="/admin/dentists">Управление врачами</Link>
```

### Шаг 3: Готово!
Откройте `/admin/dentists` и пользуйтесь админкой.

## 🎯 Функциональность

### ✅ Просмотр врачей
- Список всех зарегистрированных врачей

### ✅ Управление статусами
- **Ожидает** - новый врач, требует проверки
- **Одобрен** - врач прошел проверку
- **Отклонен** - врач не прошел проверку

- Полное удаление профиля и аккаунта

## 🔗 Backend endpoints

Админка использует эти endpoints (уже реализованы в backend):

```
GET /dentists/list - получить всех врачей
PUT /dentists/{id}/status - изменить статус
```
