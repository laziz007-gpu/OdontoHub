# Backend MVP – Dentistry App

Tech stack:
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy

Запуск проекта:
1. Создать и активировать venv
2. Установить зависимости:
   pip install -r requirements.txt
3. Запустить сервер:
   uvicorn app.main:app --reload
